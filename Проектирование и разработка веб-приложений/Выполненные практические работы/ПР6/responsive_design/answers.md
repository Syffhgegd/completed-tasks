# Ответы на вопросы по адаптивной верстке

1. **Viewport meta tag** — это `<meta name="viewport" content="width=device-width, initial-scale=1.0">`. Он сообщает браузеру, что ширина страницы должна соответствовать ширине экрана устройства, а масштаб — 1:1. Без него мобильные браузеры эмулируют десктопную ширину (~980px), и сайт выглядит уменьшенным.

2. В работе использован **Responsive подход** (адаптивный дизайн): макет гибко перестраивается под любую ширину с помощью fluid-сеток, медиа-запросов и `clamp()`. Также частично применён **Mobile First** — базовые стили ориентированы на мобильные, а улучшения добавляются для больших экранов.

3. **Mobile First** — это подход, при котором сначала пишутся стили для мобильных устройств, а затем с помощью `min-width` добавляются улучшения для больших экранов. Преимущества:  
   - Меньше переопределений,  
   - Лучшая производительность на мобильных,  
   - Фокус на контенте, а не на декоре.

4. Наиболее предпочтительные единицы:  
   - **`%`** — для ширины контейнеров и отступов,  
   - **`rem`** — для шрифтов и отступов (масштабируется с корневым шрифтом),  
   - **`em`** — для локального масштабирования (например, отступы внутри компонента),  
   - **`vw/vh`** — для full-width элементов или hero-секций.  
   Они обеспечивают гибкость и масштабируемость.

5. **`clamp(min, preferred, max)`** задаёт значение, которое:  
   - Не меньше `min`,  
   - Не больше `max`,  
   - Идеально — `preferred` (обычно с `vw`).  
   Пример: `font-size: clamp(1rem, 2.5vw, 1.5rem);` — шрифт растёт плавно, но не становится слишком маленьким или огромным.

6. Использованы брейкпоинты:  
   - **1024px** — верхняя граница планшетов,  
   - **768px** — переход к мобильной навигации,  
   - **480px** — маленькие смартфоны.  
   Значения выбраны на основе популярных устройств (iPad, iPhone, Android).

7. **Медиа-запросы** позволяют применять стили только при определённых условиях (например, ширине экрана).  
   Синтаксис:  
   ```css
   @media (max-width: 768px) {
     .container { width: 95%; }
   }
