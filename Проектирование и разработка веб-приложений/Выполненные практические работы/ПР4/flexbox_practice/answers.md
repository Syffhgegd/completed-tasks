# Ответы на вопросы по Flexbox

1. **Основное свойство — `display: flex`.** При его применении к родительскому элементу все его прямые дочерние элементы становятся flex-элементами и подчиняются правилам Flexbox (выравнивание, растяжение, порядок и т.д.).

2. **`justify-content`** управляет выравниванием по **главной оси** (по умолчанию — горизонтальной), например: `center`, `space-between`.  
   **`align-items`** — по **поперечной оси** (по умолчанию — вертикальной), например: `center`, `stretch`.  
   Пример: чтобы центрировать блок по обеим осям — `justify-content: center; align-items: center;`.

3. **`flex-direction`** задаёт направление главной оси. Возможные значения:  
   - `row` (по умолчанию, слева направо),  
   - `row-reverse` (справа налево),  
   - `column` (сверху вниз),  
   - `column-reverse` (снизу вверх).  
   Это меняет ориентацию главной и поперечной осей.

4. **`flex: 1` — это сокращение для `flex: 1 1 0%`**, где:  
   - `flex-grow: 1` — элемент может расти,  
   - `flex-shrink: 1` — может сжиматься,  
   - `flex-basis: 0%` — начальный размер до распределения свободного места.

5. **`flex-wrap: wrap`** позволяет flex-элементам переноситься на новую строку, если не помещаются в контейнер. Используется при адаптивных сетках или когда количество элементов динамическое.

6. **`align-items`** выравнивает **отдельные элементы** по поперечной оси.  
   **`align-content`** выравнивает **целые строки** (при наличии нескольких строк, т.е. при `flex-wrap: wrap`).  
   Применяется только если есть свободное место по поперечной оси и несколько строк.

7. **`order`** задаёт порядок отображения элемента (по умолчанию 0). Элементы с меньшим `order` идут первыми. Полезно для изменения порядка без изменения HTML (например, мобильная навигация).

8. **Главная ось (main axis)** — направление, заданное `flex-direction`.  
   **Поперечная ось (cross axis)** — перпендикулярна главной.  
   Например, при `flex-direction: row` — главная ось горизонтальная, поперечная — вертикальная.

9. **Адаптивность без медиа-запросов** достигается за счёт:  
   - `flex: 1` — элементы автоматически занимают доступное пространство,  
   - `flex-wrap: wrap` — элементы переносятся при нехватке места,  
   - `gap` — задаёт промежутки, не ломая layout.  
   Пример: `.container { display: flex; flex-wrap: wrap; } .item { flex: 1 1 200px; }` — колонки будут перестраиваться под ширину экрана.

10. **Преимущества `gap`:**  
    - Не влияет на внешние отступы контейнера,  
    - Автоматически не добавляет отступы снаружи,  
    - Упрощает верстку (не нужно компенсировать margin с помощью отрицательных значений),  
    - Поддерживается во всех современных браузерах.
