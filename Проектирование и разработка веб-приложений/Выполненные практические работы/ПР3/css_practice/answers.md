1. Селектор по ID (#id) имеет специфичность 100, по классу (.class) — 10. ID должен быть уникальным, класс можно использовать многократно.
2. Каскад определяет приоритет стилей: важность (!important), источник, специфичность, порядок в коде.
3. Специфичность:
   - ul li.important → 0,0,1,2
   - #site-title → 0,1,0,0
   - .content > .featured-post → 0,0,2,0
4. Наследуются: color, font-family, font-size. Не наследуются: margin, padding, border.
5. > выбирает только прямых детей, пробел — всех потомков любого уровня.
6. :hover — при наведении (интерактив), :nth-child(n) — для стилизации n-го элемента (например, зебра-таблица).
7. !important принудительно применяет правило, ломая каскад. Использовать осторожно.
8. ::before/::after вставляют виртуальный контент через content (только текст или пустая строка). Используются для декораций.
9. При одинаковой специфичности побеждает последнее объявленное правило.
10. DevTools показывают применённые стили, перечёркивают переопределённые, отображают наследование и позволяют редактировать CSS в реальном времени.
