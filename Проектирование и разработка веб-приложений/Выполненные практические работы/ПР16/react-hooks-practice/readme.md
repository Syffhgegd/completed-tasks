## Требования

- **Node.js** версии **14.x или 16.x**  
  *(Node.js 18+ может собирать код без поддержки IE11)*
- **npm** или **yarn**
- Текстовый редактор (VS Code, Notepad++, Sublime и др.)
- Браузер: Chrome, Firefox, Edge — **или IE11** (в режиме совместимости)

> Рекомендуется:  
> - В VS Code сохранять файлы в кодировке **UTF-8 with BOM** (иначе кириллица в IE11 → иероглифы)  
> - Для запуска в IE11 используйте **Edge → Меню → «Открыть в режиме Internet Explorer»**

## Как запустить проект

### 1. Установите зависимости
```bash
npm install
```

### 2. Запустите в режиме разработки
```bash
npm start
```
